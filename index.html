Grid Container và Grid Items

📚Lý thuyết

Grid Container: Là phần tử cha có thuộc tính display: grid

Grid Items: Là các phần tử con trực tiếp của Grid Container

Cách hoạt động: Khi bạn thêm display: grid vào một phần tử, tất cả con trực tiếp của nó sẽ trở thành grid items.

CSS
.container { display: grid; /* Biến thành Grid Container */ } .item { /* Tự động trở thành Grid Item */ }

💡 Điểm quan trọng:

  • Chỉ con trực tiếp mới là grid items
  • Mặc định: grid có 1 cột, items xếp theo chiều dọc
  • Grid không ảnh hưởng đến các cháu (grandchildren)

Grid Template Columns - Định Nghĩa Cột

📚Lý thuyết

grid-template-columns: Định nghĩa số lượng và kích thước các cột

Đơn vị fr (fraction): Chia không gian còn lại thành các phần bằng nhau

repeat(): Hàm lặp lại giá trị để code ngắn gọn hơn

CSS
/* 3 cột bằng nhau */ grid-template-columns: 1fr 1fr 1fr; /* Viết tắt bằng repeat */ grid-template-columns: repeat(3, 1fr); /* Cột có kích thước khác nhau */ grid-template-columns: 200px 1fr 2fr;

💡 Giải thích:

  • 1fr: Chiếm 1 phần không gian
  • 2fr: Chiếm 2 phần (gấp đôi 1fr)
  • 200px: Kích thước cố định
  • repeat(3, 1fr): Lặp "1fr" 3 lần

Gap - Khoảng Cách Giữa Items

📚Lý thuyết

gap: Tạo khoảng cách giữa các grid items (thay thế margin)

row-gap: Khoảng cách giữa các hàng

column-gap: Khoảng cách giữa các cột

CSS
/* Cách 1: Gap cho cả hàng và cột */ gap: 20px; /* Cách 2: Gap riêng */ row-gap: 20px; column-gap: 15px; /* Cách 3: Shorthand */ gap: 20px 15px; /* hàng cột */

💡 Tại sao dùng gap thay vì margin?

  • Không tạo khoảng cách ở rìa ngoài
  • Code ngắn gọn hơn
  • Dễ điều chỉnh responsive

Auto-fit và Minmax - Grid Tự Động Responsive

📚Lý thuyết

auto-fit: Tự động điều chỉnh số cột dựa trên kích thước màn hình

minmax(): Định nghĩa kích thước min và max của cột

Kết hợp: Tạo grid responsive không cần media query

CSS
/* Grid tự động responsive */ grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Giải thích: - auto-fit: Tự động tính số cột phù hợp - minmax(200px, 1fr): Mỗi cột tối thiểu 200px, tối đa 1fr - Kết quả: Tự động xuống hàng khi không đủ chỗ */

💡 Cách hoạt động:

  • Màn hình rộng: Nhiều cột
  • Màn hình hẹp: Ít cột hơn
  • Tự động xuống hàng khi < 200px
  • Không cần viết media query!

Grid Column Span - Item Chiếm Nhiều Cột

📚Lý thuyết

grid-column: Định nghĩa item chiếm bao nhiêu cột

span: Số cột mà item sẽ chiếm (kéo dài)

Ứng dụng: Tạo layout phức tạp, highlight item quan trọng

CSS
/* Grid 4 cột */ .container { display: grid; grid-template-columns: repeat(4, 1fr); } /* Item chiếm 2 cột */ .item-large { grid-column: span 2; } /* Item chiếm 3 cột */ .item-xlarge { grid-column: span 3; }

💡 Cách tính:

  • span 1: Chiếm 1 cột (mặc định)
  • span 2: Chiếm 2 cột liền kề
  • span 3: Chiếm 3 cột liền kề
  • Tự động xuống hàng nếu không đủ chỗ

Grid Template Areas - Đặt Tên Vùng

📚Lý thuyết

grid-template-areas: Đặt tên cho các vùng trong grid

grid-area: Gán item vào vùng đã đặt tên

Ưu điểm: Dễ hình dung layout, code dễ đọc

CSS
.container { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; grid-template-columns: 200px 1fr 1fr; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

💡 Quy tắc:

  • Mỗi dòng trong dấu ngoặc là 1 hàng
  • Tên giống nhau = item chiếm nhiều ô
  • Dùng dấu chấm "." cho ô trống
  • Tên phải tạo thành hình chữ nhật

Alignment - Căn Chỉnh Items

📚Lý thuyết

align-items: Căn chỉnh theo chiều dọc (vertical)

justify-items: Căn chỉnh theo chiều ngang (horizontal)

place-items: Shorthand cho cả hai (vertical horizontal)

CSS
/* Căn chỉnh dọc */ align-items: start; /* Trên */ align-items: center; /* Giữa */ align-items: end; /* Dưới */ /* Căn chỉnh ngang */ justify-items: start; /* Trái */ justify-items: center; /* Giữa */ justify-items: end; /* Phải */ /* Shorthand */ place-items: center center; /* Giữa cả 2 chiều */

💡 Ghi nhớ:

  • align- = Dọc (↕️)
  • justify- = Ngang (↔️)
  • -items = Áp dụng cho tất cả items
  • -self = Áp dụng cho 1 item riêng

Tóm Tắt CSS Grid

🎯 Những gì bạn đã học:

  • display: grid - Tạo grid container
  • grid-template-columns - Định nghĩa cột
  • gap - Khoảng cách giữa items
  • repeat() & fr - Tạo cột nhanh và linh hoạt
  • auto-fit & minmax() - Grid responsive tự động
  • grid-column: span - Item chiếm nhiều cột
  • grid-template-areas - Đặt tên vùng layout
  • align-items & justify-items - Căn chỉnh items

💪 Tip Pro: CSS Grid xuất sắc cho layout 2D (hàng + cột). Kết hợp với Flexbox (cho layout 1D) để tạo giao diện hoàn hảo!
